# MyOfficeApi

.NET Core 8 Web API CRUD Demo

## Features

- GET /api/myofficeacpd
- GET /api/myofficeacpd/{id}
- POST /api/myofficeacpd
- PUT /api/myofficeacpd/{id}
- DELETE /api/myofficeacpd/{id}

## Tech Stack

- .NET Core 8 Web API
- SQL Server
- Swagger
- ADO.NET
